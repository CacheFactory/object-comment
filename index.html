Object-Comment


Easily add comments to any Backbone.js model
As used in Process Smith

Project Setup

  • Add the following libraries: jQuery, bootstrap, underscore.js, backbone.js, and this bootstrap fork
  • Your project needs a backend that can save the Comment model
  • Your project needs to implament a Person model

Usage examples

  • Comment button

    Attach this button to a list of objects.



    var commentButtonView= new ThreadView(
	{el:$('#commentButton'),
	objectName: 'AnyObject',
	objectId: 1});

  • Comment list

    Creates a list of comments.



    var commentBoxView= new CommentBoxView(
	{el:$('#commentButton'),
	objectName: 'AnyObject',
	objectId: 1});
